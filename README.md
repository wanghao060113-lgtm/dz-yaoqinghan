# 物理与电子工程学院2025级新生迎新晚会暨表彰大会电子邀请函

## 项目简介

这是一个 H5 电子邀请函项目，采用单页应用（SPA）架构，包含5个页面，支持在手机 QQ/微信中打开浏览。采用国潮风格设计，包含雪花动画、背景音乐、节目单展示、排练风采展示等功能。

## 项目结构

```
电子邀请函/
├── index.html          # 主页面（5页SPA应用）
├── styles.css          # 全局样式文件（国潮风格）
├── script.js           # JavaScript 文件（雪花效果、音乐管理、页面切换）
├── package.json        # 项目配置文件
├── vercel.json         # Vercel 部署配置
├── .gitignore          # Git 忽略文件配置
├── public/
│   └── bgm.mp3         # 背景音乐文件
└── photos/
    └── gallery/        # 排练照片目录
        ├── p1.1.jpg
        ├── p1.2.jpg
        ├── p2.1.jpg
        ├── p2.2.jpg
        ├── p3.1.jpg
        └── p3.2.jpg
```

## 页面结构

1. **第1页：封面页** - 欢迎页面，包含雪花动画和进入按钮
2. **第2页：正式邀请函** - 活动详情和邀请内容
3. **第3页：节目单** - 完整节目列表，包含参演人员信息
4. **第4页：排练风采展示** - 排练照片展示
5. **第5页：感谢页** - 结束页面

## 功能特点

- ✅ 5页单页应用（SPA）架构，音乐持续播放不中断
- ✅ 点击第一页按钮时立即开始播放音乐
- ✅ 雪花粒子效果（使用 ❄ 字符，性能优化）
- ✅ 国潮风格UI设计，包含渐变背景和装饰元素
- ✅ 响应式设计，完美适配手机端（QQ/微信）
- ✅ 右上角音乐开关按钮，可控制播放/暂停
- ✅ 节目单展示，支持查看参演人员详情（弹窗）
- ✅ 排练照片展示，支持懒加载和错误处理
- ✅ 页面切换动画，流畅的用户体验
- ✅ 底部进度指示器，显示当前页面位置
- ✅ 支持 Vercel 和 GitHub Pages 部署

## 部署说明

### Vercel 部署

1. 将项目推送到 GitHub 仓库
2. 在 Vercel 中导入项目
3. 配置域名（如：romantic-hao.fun）
4. 部署完成

### GitHub Pages 部署

1. 将项目推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择主分支作为源
4. 访问 `https://username.github.io/repository-name`

## 性能优化

- ✅ 使用 `transform` 替代 `left/top` 进行动画，减少重排
- ✅ 图片懒加载（`loading="lazy"`）
- ✅ 移动端减少粒子数量，提升流畅度
- ✅ 使用 `will-change` 属性优化动画性能
- ✅ 移动端减少 `backdrop-filter` 模糊强度

## 注意事项

1. 背景音乐文件 `public/bgm.mp3` 需要自行添加
2. 所有资源路径使用相对路径，确保部署后正常访问
3. 项目采用单页应用架构，不进行页面跳转，确保音乐持续播放
4. 建议压缩图片和音频文件以提升加载速度
